1. Research for and describe the ways to open developers console in Mozilla Firefox and Safari.

Opening Developer Console in Mozilla Firefox:

- Method 1: Use the keyboard shortcut "Ctrl + Shift + I" (or "Cmd + Opt + I" on macOS) to access the Developer Tools panel and navigate to the Console tab.

- Method 2: Click on the three horizontal lines in the top-right corner to access the menu, then select "Web Developer" and click on "Web Console".

Opening Developer Console in Safari:

- Method 1: Use the keyboard shortcut "Option + Cmd + C" to access the Developer Tools panel and locate the Console tab.

- Method 2: Click on "Develop" in the menu and select "Show Error Console".

2. What way of including internal links into the HTML page you would consider generally preferable: absolute or relative links? Describe use cases for choosing absolute links and choosing relative links.

Generally, it is preferable to use relative links over absolute links for internal links:

- Relative links: are based on the location of the current page within the directory structure. They are shorter and more flexible, making them easier to handle and keep up. These links are commonly used to connect to other pages on the same site.

- Absolute links: on the other hand, contain the entire URL, including the protocol (e.g., http://) and domain. They are necessary when linking to external sites or when the page structure may change.

Use cases for choosing absolute links:

- To link to external websites or pages outside your domain.

- When the page structure may change often, and you want to make sure the link stays valid.

Use cases for choosing relative links:

- To link to other pages within the same site.

- When the website structure is expected to remain stable, making it easier to maintain links.

- When the page is stored locally and not yet live on a server.

3. Research and write about image formats used in web-development. What is the difference between JPG, GIF and PNG formats? What is WEBP image type? What is SVG?

- JPG (or JPEG): It is a type of compression format often used for photos or detailed images with many colors. JPGs can display millions of colors and have smaller file sizes, but they may lose some image quality.

- GIF: It is a compression format commonly used for simple animations and images with a limited color range. GIFs can show transparency and animations but have fewer color options.

- PNG: It is a compression format often used for images with transparent backgrounds. PNGs maintain image quality but can have larger file sizes compared to JPGs.

- SVG: It stands for Scalable Vector Graphics. SVG is a type of image format that uses XML-based code to describe graphics. Unlike traditional images (JPG, GIF, PNG), SVGs can be resized without losing quality. They are ideal for logos, icons, and other graphics that need to be flexible in size and resolution. SVGs can be animated and tweaked using CSS and JavaScript, too.
